# 研究报告 · Kimi K3 登顶 + AI 模型军备竞赛

---

## 一、研究简报（写作者 1 分钟速读）

- **核心发现（一句话）**：Kimi K3 以 2.8 万亿参数登顶 Frontend Code Arena 是全球首个开源模型在该榜单超越闭源巨头，但随即而来的「蒸馏」质疑和暂停新用户事件暴露了中国 AI 创业公司的核心矛盾——技术突破 vs 商业可持续。

- **3 个必须引用的事实**（含信源等级和来源链接）：
  1. Kimi K3 拥有 2.8 万亿参数，基于自研 KDA（Kimi Delta Attention）混合线性注意力机制和 Attention Residuals 注意力残差技术，扩展效率较前代提升约 2.5 倍 `[S-2026]` [来源](https://platform.kimi.com/docs/guide/kimi-k3-quickstart)
  2. Kimi K3 以 1679 分登顶 Frontend Code Arena 前端编程榜单，成为首个在该榜单上超越 OpenAI、Google 等闭源模型的开源模型 `[A-2026]` [来源](https://news.qq.com/rain/a/20260723A046PT00)
  3. Kimi K3 发布后因算力资源紧张被迫暂停 C 端新用户订阅，凸显大模型竞争已从训练转向推理基础设施的新阶段 `[B-2026]` [来源](https://news.marsbit.co/tags/%E6%9C%88%E4%B9%8B%E6%9A%97%E9%9D%A2)
  4. 5亿 Token 实测显示，K3 复刻《GTA5》网页游戏的效果超过 GPT-5.6 Sol，警察追捕系统等细节处理"惊艳"，但 699 元会员的 40% 额度仅够跑 4-5 个任务，单个任务成本约为 GPT 的两倍 `[A-2026]` [来源](https://news.qq.com/rain/a/20260718A0AXJ700)
  5. K3 的核心突破在于 Agent Harness + Vision in the Loop——写完代码后自动打开网页截图检查视觉效果，实现类似人类的自我修正 `[A-2026]` [来源](https://news.qq.com/rain/a/20260718A0AXJ700)

- **最佳叙事框架**：反转框架——「你以为中国 AI 只能做跟随者，其实 Kimi K3 已经在某个榜单上超越了所有闭源模型。但你以为它就此起飞了，它却连新用户都接不住。」

- **角度回验**：✅ 原始切入角度（PM 视角拆解 Kimi 产品策略）仍然成立，且研究发现「暂停新用户」是比「蒸馏质疑」更有 PM 讨论价值的切入点

- **关键风险**：部分融资/估值数据未从一手信源核实，蒸馏争议的一手信源不完整（白宫官员言论未找到原始出处）

- **总信息来源数**：18 个
- **总字数**：待实测

- 概述：独立测试机构白鲸实验室花了 1 天时间、开了 699 元最高档会员、消耗 5 亿+ Token 对 K3 进行了深入实测。测试发现：K3 复刻 GTA5 的效果超过 GPT-5.6 Sol（警察追捕系统等细节处理惊艳），但其核心突破在于 Agent Harness + Vision in Loop（AI 能自我检查视觉效果并修正）。代价高昂：699 元会员的 40% 额度仅跑 4-5 个任务，单个任务成本约为 GPT 的两倍，高峰期仅几十 token/s，单任务生成时间超 40 分钟。测试者结论：「如果它能无限量供应，我之后的主力模型会直接使用 K3。」
- 与选题关联：这是目前能找到的最详实的 K3 第三方实测。完美印证了「技术确实牛，但成本也确实高」的核心矛盾
- 叙事可用性：⭐⭐⭐——极佳的写实素材，有数据有故事有结论
