## 一、研究简报（写作者 1 分钟速读）

- **核心发现（一句话）**：在已有 12 个热榜源之外，至少还有 6 个高价值的可爬取热榜源值得接入 —— IT之家、澎湃新闻、网易新闻、InfoQ 中国、虎嗅、品玩
- **3 个你必须在适配器中优先接入的事实**：
  1. **网易新闻** 和 **InfoQ 中国** 的热榜自带热度数值（阅读量/互动量），可直接用于融合榜单热度计算

- **最佳叙事框架**：分类对比框架 — 按 IT 互联网 / AI / 时事政治 / 历史政治 四个领域分别列出推荐源
- **角度回验**：✅ 角度成立，研究发现了多个优质未接入热榜源
- **关键风险**：部分平台（观察者网）有 CDN 防护无法直连；部分平台热榜条数少（品玩仅5条）

### 三、高潜力平台逐项分析

#### 1. IT之家（ithome.com）⭐⭐⭐

- **类型**：IT 科技新闻门户
- **热榜位置**：首页右侧「日榜 / 周榜 / 月榜」三标签
- **URL**：`https://www.ithome.com/`（热榜嵌入首页 HTML）
- **提取方式**：解析首页 HTML 中排行榜区域 `<ul>/<li>` 列表
- **数据结构**：排名序号 1-12 + 文章标题 + URL（`https://www.ithome.com/0/{分类}/{id}.htm`）
- **条目数**：12 条（日榜 12 条，周/月榜各 12 条，共 36 条）
- **热度指标**：无独立热度值，但排名本身即热度排序
- **需登录**：否
- **特点**：IT之家是中文互联网最大的 IT 数码新闻门户，日榜覆盖当日最热 IT 话题，**强烈推荐首批接入**
- **今日热榜示例**：我魂都吓没了（AWS万亿账单）、腾讯WorkBuddy发布、华为畅享90 Pro Max激活310万台

#### 2. 澎湃新闻（thepaper.cn）⭐⭐⭐

- **类型**：时事政治新闻平台
- **热榜位置**：首页中部「热榜」模块
- **URL**：`https://www.thepaper.cn/`（热榜嵌入首页 HTML）
- **提取方式**：首页 HTML 解析，编号 1-20
- **数据结构**：序号（1-20）+ 标题 + 文章链接（`newsDetail_forward_XXX`）
- **条目数**：20 条（是所有来源中条目最多的）
- **热度指标**：无独立热度值
- **需登录**：否
- **特点**：澎湃新闻是时政类头部媒体，20 条热榜覆盖面广，适合时事政治类选题
- **可爬性**：高，静态 HTML 直接渲染

#### 3. 网易新闻（news.163.com）⭐⭐⭐

- **类型**：综合新闻门户
- **热榜位置**：首页「热点排行」模块
- **URL**：`https://news.163.com/`（首页 HTML）
- **提取方式**：首页 HTML 解析，定位「热点排行」区域
- **数据结构**：序号 1-10 + 标题 + 链接 + **热度数值**（如 23025）
- **条目数**：10 条
- **热度指标**：✅ 有（疑似浏览量/互动量）
- **需登录**：否
- **特点**：**唯一一个自带热度数值的时政新闻热榜**，热度值可直接用于融合榜单

#### 4. InfoQ 中国（infoq.cn）⭐⭐⭐

- **类型**：企业级 IT 技术媒体
- **热榜位置**：独立热榜页面
- **URL**：`https://www.infoq.cn/hotlist?tag=day`（支持 `?tag=day` / `?tag=month` / `?tag=quarter`）
- **提取方式**：独立热榜页 HTML 解析，或 API 接口
- **数据结构**：标题 + 摘要 + 作者 + 时间 + **阅读量数值**（如 8360）
- **条目数**：10+ 条（可滚动加载更多）
- **热度指标**：✅ 有（阅读量数值）
- **需登录**：浏览不登录可见，部分详情需登录
- **特点**：**唯一有专用热榜URL+时间切换+热度数值的IT技术平台**，强烈推荐

#### 5. 虎嗅（huxiu.com）⭐⭐

- **类型**：科技商业媒体
- **热榜位置**：首页「48小时热文」模块
- **URL**：`https://www.huxiu.com/`（首页 HTML）
- **提取方式**：首页 HTML 解析 48小时热文区域
- **数据结构**：文章标题 + URL（`/article/{id}.html`）+ 作者 + 评论数 + 发表时间
- **条目数**：约 10+ 条
- **热度指标**：无排名数字，但有评论数可作为热度参考
- **需登录**：否
- **特点**：虎嗅是科技商业领域头部媒体，48小时热文反映科技圈关注焦点

#### 6. 品玩（pingwest.com）⭐⭐

- **类型**：科技媒体
- **热榜位置**：首页「一周精选」模块
- **URL**：`https://www.pingwest.com/`（首页 HTML）
- **提取方式**：首页 HTML 解析一周精选区域
- **数据结构**：排名 1-5 + 标题 + 作者 + 评分摘要
- **条目数**：5 条
- **热度指标**：无热度值
- **需登录**：否
- **特点**：量少但编辑精选含金量高，适合作为选题参考
